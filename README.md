# 🚗 Smart Bluetooth Controlled Car (ESP32)

## 📌 Description
This project is a feature-rich Bluetooth-controlled smart car built using ESP32. It can be controlled wirelessly via a mobile application and supports multiple functionalities including movement control, lighting system, and horn.

---

## 🚀 Features
- 📱 Bluetooth-based mobile control  
- 🔄 Movement control:
  - Forward  
  - Backward  
  - Left  
  - Right  
- 💡 Front Light control  
- 🔴 Back Light control  
- ⚠️ Hazard Light (all indicators blinking)  
- 🔊 Horn / Buzzer system  
- ⚡ Real-time response  

---

## 🛠️ Technologies Used
- ESP32 Microcontroller  
- L293D Motor Driver  
- DC Motors  
- Bluetooth Communication  
- LEDs (Front, Back, Hazard)  
- Buzzer  
- Arduino IDE  

---

## ⚙️ How It Works
- Mobile app sends commands via Bluetooth  
- ESP32 receives signals and processes them  
- Motor driver controls movement  
- LEDs and buzzer are triggered based on commands
